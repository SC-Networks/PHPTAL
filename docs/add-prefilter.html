addPreFilter() method

Note

In PHPTAL 1.2.1 this method replaced older setPreFilter() method, which is now deprecated.

Adds new pre filter that will be applied to the template. Pre filters can modify source code and parsed DOM nodes of the template. Pre filters are applied in order they've been set.

See description of PHPTAL_PreFilter for more information how to write your own pre filter.
